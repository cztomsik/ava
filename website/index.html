Frequently Asked Questions

What is Ava?

Ava is a personal language server that runs locally on your computer. It allows you to process language tasks without sharing your data online.

What is a language server?

A language server is a program that processes language tasks, such as spell checking, grammar checking, and more.

What is a language model?

A language model is a program that processes language tasks, such as spell checking, grammar checking, and more.

How do I use Ava?

Ava is a desktop application. Simply download and install Ava, and you're ready to go.

What operating systems does Ava support?

Ava supports Windows and macOS. Linux support is planned, but there is no ETA yet.

What languages does Ava support?

It depends on the language model you use. Ava supports any language model that you can install.

How much does Ava cost?

Ava is free to use. We may offer premium features in the future, but Ava will always be free to use.
