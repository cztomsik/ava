Frequently Asked Questions

What is a language server?

A language server is a specialized program that processes language-related tasks. This includes activities like text generation, grammar correction, rephrasing, summarization, data extraction, and more.

What is a language model?

A language model is a type of artificial intelligence that understands and generates human language. It's the engine behind many modern language processing tools.

How is privacy ensured?

Ava processes all language tasks locally on your device. This means your data never leaves your computer and is not shared with any external servers, ensuring maximum privacy.

Do I need an internet connection?

Once Ava is downloaded and set up, you don't need an active internet connection. However, to download a language model, you'll need to be connected to the internet.

How much does it cost?

Good news! The basic version is completely free. There will be a premium version in the future, but the core functionalities will always remain accessible without any cost.

How do I get started?

It's simple! Download Ava, install it on your desktop, and you're all set to explore its capabilities.

What languages are supported?

It depends on the language model you choose. Theoretically, any language can be supported. However, majority of models are trained on English. This will likely improve in the future but it's not something we can control.

Which operating systems are supported?

Currently, only macOS version is available. Windows is coming soon and Linux is planned for the future.

Is there a version for mobile devices?

Not at the moment. However, we're exploring the possibility of a mobile version. Stay tuned for updates!
